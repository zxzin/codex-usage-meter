# Token Meter App Review Sample

This sample lets App Review inspect Token Meter without receiving a real Codex `auth.json` or account token.

1. Download and extract `token-meter-review-sample.zip`.
2. Start the 7-day trial in the App Store sandbox if prompted.
3. When Token Meter asks for the Codex folder, select the extracted `Token Meter Review Sample` folder.
4. The widget displays sample 5-hour and weekly quota values and continuously animates the bees at a representative token burn rate.

The sample `auth.json` contains only the explicit review marker. It contains no username, password, access token, refresh token, account identifier, or other credential. The review path is enabled only in the Mac App Store build.
